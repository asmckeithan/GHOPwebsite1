To publish: save this file as index.html and push it to a GitHub repo, or drag & drop to a free host. See the notes below for quick options.

Instructions: Using VS Code & Publishing

  1. Install VS Code: Download Visual Studio Code from code.visualstudio.com.
  2. Open your folder: Launch VS Code ‚Üí File ‚Üí Open Folder ‚Üí choose your project folder.
  3. Edit the site: Create or open index.html and make your changes. Save automatically with Ctrl+S (Cmd+S on Mac).
  4. Preview: Install the 'Live Server' extension ‚Üí Right-click the HTML file ‚Üí 'Open with Live Server' to preview in your browser.
  5. Add JavaScript: Create a new file named script.js and link it before the closing </body> tag using <script src="script.js"></script>.
  6. Publish via GitHub Pages:
    • Create a GitHub account (if needed).
    • Create a new public repository.
    • Upload your index.html.
    • Go to Settings ‚Üí Pages ‚Üí Source ‚Üí choose 'main branch' ‚Üí Save.
    • Visit https://yourusername.github.io/repositoryname to see your live site.
  7. Publish via Netlify (no GitHub needed):
    • Go to netlify.com.
    • Sign up and log in.
    • Drag & drop your index.html file into the dashboard 'Sites' area.
    • Netlify automatically hosts it with a free subdomain (e.g., yourname.netlify.app).
